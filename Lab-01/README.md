# Lab 01 - HTML CV

## 📌 Objective
Create a plain CV (Curriculum Vitae) using only HTML.

## 🎯 Requirements
- ❌ No CSS
- ❌ No JavaScript
- ✅ Use semantic HTML tags only

## 📄 File
- `cv.html` - Main CV file

## 📋 CV Sections
1. Name and Title
2. Contact Information
3. Objective
4. Education
5. Skills (Technical & Soft Skills)
6. Experience
7. Hobbies & Interests

## 🏷️ HTML Tags Used
- `html`, `head`, `body`
- `h1`, `h2`, `h3`
- `p`, `ul`, `li`
- `table`, `thead`, `tbody`, `tr`, `th`, `td`
- `section`, `article`, `header`, `footer`
- `hr`, `br`
- `strong`, `em`

## 🚀 How to View
Simply open `cv.html` in any web browser.
